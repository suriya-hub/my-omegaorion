# React + Vite Project

## Setup Instructions

1. Clone the repository  
   `git clone https://github.com/suriya-hub/my-omegaorion`

2. Go to the project directory  
   `cd <project-folder>`

3. Install dependencies  
   `npm install`

4. Run the development server  
   `npm run dev`

5. Open the displayed local URL in your browser

---

## Tech Stack Used

- React.js (with Vite)
- Tailwind CSS (for styles)

---

## Assumptions Made

- Node.js and npm are already installed
- Runs in a modern browser
- Uses mock/static data (no backend)

---

## Available Scripts

- `npm run dev` → Start development server  
- `npm run build` → Create production build  
- `npm run preview` → Preview production build
